## 개발 언어

 - front-end
	 - Vue.js
 - Back-end
	 - Spring Boot
	 - AWS EC2



# 시작하기

### 실행환경

- yarn
- npm

### 설치

프로젝트를 클론합니다.

```
git clone https://github.com/tmdgjs/blended-codingtest
```

yarn 패키지를 설치합니다.

```
yarn
```

### 프론트엔드 개발 서버 가동

클론했던 디렉터리로 이동해 개발 서버를 실행합니다.

```
npm run serve -- --port 3000
```

http://localhost:3000/ 으로 접속하면 실행 가능합니다.
